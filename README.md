3D Cylon Raider from Battlestar Gallactica rendered in WebGL using HTML5 and Babylon.JS. Note: this will autoplay BSG theme music upon load.

![App Screenshot](https://github.com/saraalfadil/3D-Cylon-Raider/blob/master/raider.gif?raw=true)

## Run the project

Run local server with Python:

```
python -m http.server 8000 
```

Then open http://localhost:8000 in your browser.
